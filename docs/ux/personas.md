# Personas — SPU SE Teacher Platform

## อาจารย์ประจำ (Primary)

- **อายุ:** 30–55
- **บทบาท:** สอนและดูแลนักศึกษา คณะการสร้างเจ้าของธุรกิจ
- **อุปกรณ์:** โน้ตบุ๊กใน office, แท็บเล็ต/มือถือในห้องเรียน
- **เป้าหมาย:** เปิดเครื่องมือสอนได้เร็ว (Word Cloud, Poll, เช็กชื่อ) โดยไม่ต้องเรียนรู้ระบบซับซ้อน
- **Pain points:** เมนูลึกเกินไป, sidebar popup บัง content, หาเครื่องมือจาก 48 รายการยาก

## ผู้ช่วยสอน / TA (Secondary)

- **บทบาท:** ช่วยเช็กชื่อ, ติดตามคะแนน, ประสานงานโครงการ
- **เป้าหมาย:** เข้าหมวดที่รับผิดชอบได้ตรง (เช็กชื่อ, คะแนน, โครงการ)
- **Pain points:** ไม่รู้ว่าเครื่องมืออยู่หมวดไหน

## ข้อกำหนด UX จาก Persona

1. ค้นหาเครื่องมือจาก header ได้ทุกหน้า
2. Sidebar แสดงแค่หมวด — ไม่ซ่อน 48 รายการใน popup
3. URL แชร์ได้ (`/teacher/classroom/wordcloud`)
4. Mobile: drawer เมนู + grid 1 คอลัมน์
