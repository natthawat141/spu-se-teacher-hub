# User Flows

## Flow 1 — หาเครื่องมือด้วย Search

1. อาจารย์เปิด `/teacher`
2. พิมพ์ใน search bar header (เช่น "word")
3. เลือกผลลัพธ์ → ไป `/teacher/classroom/wordcloud`

## Flow 2 — เข้าหมวดจาก Sidebar

1. คลิก "กิจกรรมในชั้นเรียน" ใน sidebar
2. ไป `/teacher/classroom`
3. เห็น grid 13 เครื่องมือ
4. คลิก "Word Cloud" → `/teacher/classroom/wordcloud`

## Flow 3 — เข้าหมวดจาก Dashboard

1. อยู่ `/teacher`
2. คลิกการ์ดหมวด "เช็กชื่อเข้าเรียน"
3. ไป `/teacher/attendance`
4. เลือกเครื่องมือ

## Flow 4 — Mobile

1. กด ☰ → Drawer เปิด
2. เลือกหมวด → Drawer ปิด → หน้าหมวด
3. หรือใช้ search แทน sidebar
