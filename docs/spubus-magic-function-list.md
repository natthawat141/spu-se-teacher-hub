# SPUBUS MAGIC — รายการฟังก์ชัน/เครื่องมือ (จากการสแกนเว็บจริง)

แหล่งที่มา: สำรวจจากเว็บ `spubus-teacher-support.vercel.app` หลังล็อกอิน (UI + route + request ที่เห็นได้)

> หมายเหตุ: รายการนี้อัปเดตได้เรื่อย ๆ — ถ้าเครื่องมือใด “ยังไม่ได้เปิดหน้าเข้าไปดู” จะระบุไว้ และอธิบายจากชื่อ/ตำแหน่งก่อน จากนั้นค่อยยืนยันด้วยการเปิดหน้าเครื่องมือนั้นจริง

## ภาพรวมระบบ
- **หน้าแรก/แดชบอร์ด**: เลือกหมวดเครื่องมือ (แสดงจำนวนเครื่องมือแต่ละหมวด)
- **เมนูภาษา**: `ไทย ▼`
- **เมนูผู้ใช้**: แสดงชื่อผู้ใช้ + มี **ออกจากระบบ**

## จำนวนฟังก์ชัน (เครื่องมือ)
จากเมนูหลังล็อกอิน ระบบแบ่งเป็น **11 หมวด** รวมทั้งหมด **49 เครื่องมือ** (13+4+1+1+7+5+2+7+7+1+1)

## หมวดเครื่องมือทั้งหมด (ตามเมนูซ้าย)

### 🎉 กิจกรรมในชั้นเรียน (13)
1) **Word Cloud**: เปิดห้องให้นักศึกษาส่งคำเข้า “กลุ่มคำ” แล้วแสดงเป็น Word Cloud (มี QR/ลิงก์, ตั้งเวลา, สร้างห้องใหม่)
2) **Live Poll**: สร้างคำถาม + ตัวเลือก เพื่อให้นักศึกษาโหวตแบบเรียลไทม์ (มี QR/ลิงก์, ตั้งเวลา, เริ่ม Poll)
3) **Brainstorm Board**: กระดานระดมความคิด ให้นักศึกษาส่ง “ไอเดีย” เข้าบอร์ด (มี QR/ลิงก์, ตั้งเวลา, เพิ่มไอเดีย, เลือกสีการ์ด/โพสต์)
4) **วงล้อสุ่ม**: สุ่มชื่อ/คำถาม/หัวข้อด้วยวงล้อ (ยังไม่ได้เปิดหน้าละเอียด)
5) **กล่องสุ่ม**: สุ่มของรางวัล/คำตอบ/ภารกิจ (ยังไม่ได้เปิดหน้าละเอียด)
6) **สุ่มกลุ่มนักเรียน**: จัดกลุ่มนักศึกษาแบบสุ่ม (ยังไม่ได้เปิดหน้าละเอียด)
7) **กระดานคะแนนทีม**: คะแนนทีม/แข่งขันในห้องเรียน (ยังไม่ได้เปิดหน้าละเอียด)
8) **สร้างแบบทดสอบ**: สร้าง quiz (ยังไม่ได้เปิดหน้าละเอียด)
9) **สร้างแบบทดสอบวิดีโอ**: สร้าง quiz แบบแนบ/อิงวิดีโอ (ยังไม่ได้เปิดหน้าละเอียด)
10) **ทดสอบก่อน-หลังเรียน**: เปรียบเทียบผลก่อนเรียน/หลังเรียน (ยังไม่ได้เปิดหน้าละเอียด)
11) **เกมตอบคำถามสด**: เกมถามตอบแบบเรียลไทม์ (ยังไม่ได้เปิดหน้าละเอียด)
12) **BUS-SANOOK**: เกม/กิจกรรมรูปแบบพิเศษ (ยังไม่ได้เปิดหน้าละเอียด)
13) **งูและบันได**: เกมงูและบันไดเพื่อกิจกรรมชั้นเรียน (ยังไม่ได้เปิดหน้าละเอียด)

#### รายละเอียดที่ยืนยันจากหน้าจอจริง: Word Cloud
- **ตั้งเวลา/โหมดห้อง**: เลือกเวลา (∞/1/3/5/10/15/20 นาที)
- **เปิดห้อง + QR**: แสดง **Room Code** และลิงก์ให้นักศึกษาเข้าร่วม (เช่น `/student/interactive?room=...&mode=wordcloud`) + ปุ่มคัดลอก
- **ควบคุมห้อง**: มีปุ่มพัก/รีเซ็ต/ซ่อน QR/สร้างห้องใหม่
- **จัดการคำ**: ช่องพิมพ์คำ + ปุ่ม “เพิ่มคำ” + ปุ่ม “ตัวอย่าง”
- **API ที่พบว่าเกี่ยวข้อง**: `/api/teacher/interactive` และ `/api/teacher/interactive?room=<CODE>&action=get`

#### รายละเอียดที่ยืนยันจากหน้าจอจริง: Live Poll
- **ตั้งคำถาม**: ช่อง “คำถาม Poll *” (placeholder: “เช่น คุณคิดว่าวิธีใดเหมาะสมที่สุด?”)
- **ตัวเลือกโหวต**: ช่อง “ตัวเลือก 1..4” มีปุ่มลบ ✕ และปุ่ม `+ เพิ่มตัวเลือก`
- **เริ่มโหวต**: ปุ่ม `🗳️ เริ่ม Poll`
- **เปิดห้อง + QR**: ปุ่ม `📱 เปิดห้อง + QR` สร้างลิงก์ให้นักศึกษาเข้าร่วมรูปแบบ `/student/interactive?room=<CODE>&mode=poll` + ปุ่มคัดลอก
- **ตั้งเวลา**: dropdown เวลา (∞/1/3/5/10/15/20 นาที)

#### รายละเอียดที่ยืนยันจากหน้าจอจริง: Brainstorm Board
- **เพิ่มไอเดีย**: ช่อง “พิมพ์ไอเดีย...” + ปุ่ม `+ เพิ่ม` + ปุ่ม `📌 ตัวอย่าง`
- **เลือกสีโพสต์/การ์ด**: มีแถบสีให้เลือกหลายสี
- **เปิดห้อง + QR**: ปุ่ม `📱 เปิดห้อง + QR` สร้างลิงก์ให้นักศึกษาเข้าร่วมรูปแบบ `/student/interactive?room=<CODE>&mode=brainstorm` + ปุ่มคัดลอก
- **ตั้งเวลา**: dropdown เวลา (∞/1/3/5/10/15/20 นาที)

#### รายละเอียดที่ยืนยันจากหน้าจอจริง: วงล้อสุ่ม
- **รายการสำหรับสุ่ม**: กล่องข้อความ “ใส่ชื่อหรือรายการ (หนึ่งบรรทัดต่อหนึ่งรายการ)” (มีตัวอย่างรายชื่อให้)
- **จัดการรายการ**: ปุ่ม `✅ บันทึกรายการ`, ปุ่ม `🔄 รีเซ็ต (N รายการ)`
- **ตัวเลือก**: checkbox “ลบรายการที่ถูกเลือกออกหลังหมุน”
- **สุ่มผลลัพธ์**: ปุ่ม `🎯 หมุน!` และแสดง “วงล้อสุ่ม” ด้านขวา

## Endpoint ที่พบจากการสแกน

### สรุปตัวเลข
- **API path ฐาน (ไม่ซ้ำ) ที่ยืนยันจาก bundle/network:** **4**
- **ยังมี endpoint เพิ่มเมื่อเปิดเครื่องมืออื่น** — ต้องสำรวจด้วย HAR หรือคลิกครบ 49 เครื่องมือ

### รายการที่ยืนยันแล้ว
| Path | หมายเหตุ |
|------|----------|
| `POST /api/auth/register` | พบใน JS bundle (หน้า login) |
| `GET /api/teacher/bussanook?room=__PING__` | ping/health |
| `/api/teacher/interactive` | ห้อง Word Cloud / Poll / Brainstorm |
| `GET /api/teacher/interactive?room=<CODE>&action=get` | ดึงสถานะห้อง |
| `/api/broadcast` | พบใน bundle; เรียก `GET` ตรงได้ 404 |

### ฝั่งนักศึกษา (route สำคัญ)
- `/student/interactive?room=<CODE>&mode=wordcloud|poll|brainstorm`
